My first Java app!!

This app allows user to order coffee online, with options to add toppings. User can plus or minus number of cups to order, mention his/her name. It also shows toast message when quantity is 0 or user orders more than 100 cups at once. When click on Order button, app redirects user to mail app, where user is shown summary of order in email body.

Pretty basic right! No fancy animations or cutting edge modern UI. Yes, but hey, this was my very first app which I created and this app helped me to understand and introduced me to lot of necessary basic and core concets [UPDATE : I am still working on this app and will improve it gradually like adding splash screen and revising UI elements]
  • Foundation concepts of XML, Android specific Java
  • Plan and build layout for App UI
  • Linear and Relative layout, nested layouts (views)
  • Make app interactive and process user input
  • Event handling
  • Intents
  • App localization (multi-lingual support, xliff)
  • Toast
  • Working with styles
